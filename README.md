# Think in Japanese Portal

A retro 80s pixel-art Japanese learning portal built as a fully offline-capable static website.

The site combines kana practice, phrase study, a 90-day study calendar, reading aids, and a bilingual English/Spanish learning experience.

## Features

- Fully usable offline (no server or internet connection required)
- Pixel-art pastel theme inspired by Tokyo, Odaiba, Rainbow Bridge, and Kagoshima
- Responsive design for desktop, TV, tablet, phone, and foldable devices
- Kana Dojo practice game
- 90-day Japanese study tracker
- Reading aids and phrase decks
- English and Spanish versions
- Static HTML/CSS/JavaScript
- No login, account, backend, or database required

## Offline Friendly

One of the primary goals of this project is to make Japanese study available anywhere, even without an internet connection.

The entire portal is built using static HTML, CSS, JavaScript, and local assets. No database, backend server, account, login, API key, or internet connection is required after downloading the files.

You can:

- Download the repository
- Extract the files locally
- Open `index.html`
- Study completely offline

This makes the portal ideal for:

- Airplane travel
- Commutes
- School or work breaks
- Tablets without internet access
- Portable study drives
- Long-term personal archives

All progress, settings, and game history are stored locally in your browser when supported.
